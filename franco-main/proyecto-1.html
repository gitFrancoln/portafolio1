Proyecto E-commerce (DEMO)

Aplicación web de e-commerce desarrollada con Java y Spring Boot en el backend, MySQL como base de datos, y tecnologías de frontend como HTML, CSS, JavaScript, Thymeleaf y Bootstrap.

Ver Proyecto

Descripción detallada del proyecto

La plataforma permite a los usuarios registrarse, iniciar sesión y gestionar su cuenta, mientras que los administradores pueden crear, editar y listar productos con detalles como tallas, stock, precio y descripción. Los usuarios pueden añadir productos al carrito de compras, visualizar los detalles, y completar un formulario de envío antes de realizar el pago a través de la integración con Mercado Pago usando el checkout Bricks. El diseño es completamente responsive gracias a Bootstrap, ofreciendo una experiencia fluida en cualquier dispositivo. Git se utilizó para el control de versiones y gestión del código fuente. La aplicación está pensada para ser intuitiva, con un enfoque en la gestión eficiente de productos y compras, todo dentro de una arquitectura robusta y escalable.

Aunque la aplicación actualmente no utiliza microservicios, está diseñada con una arquitectura escalable y con la visión de ser mejorada en el futuro, permitiendo su evolución hacia una solución más robusta y distribuida.
