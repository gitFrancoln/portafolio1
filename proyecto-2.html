Aplicación Móvil

En este proyecto desarrollé una aplicación móvil diseñada para proporcionar información meteorológica precisa y en tiempo real. Desarrollada con Flutter y el lenguaje Dart, esta aplicación aprovecha la API de OpenWeatherMap para ofrecer datos detallados sobre el clima en cualquier ubicación del mundo.

Ver Proyecto

Descripción detallada del proyecto

La aplicación permite consultar las condiciones climáticas actuales, incluyendo temperatura, sensación térmica, humedad, velocidad del viento y especificaciones sobre el cielo. Además, utiliza el GPS del dispositivo para obtener datos basados en la ubicación actual. También incluye un pronóstico extendido para los próximos días, lo que la hace ideal para planificar actividades con anticipación. Es visualmente moderno y atractivo gracias a utilizar Dart y Flutter. En este proyecto también utilizó la API de OpenWeatherMap para la obtención de datos meteorológicos.

Herramientas utilizadas:

Dart
Flutter
OpenWeatherMap
Integración con GPS
