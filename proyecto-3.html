Descripción detallada del proyecto

Aplicación web diseñada como una tienda en línea donde los usuarios pueden explorar un catálogo de productos, agregarlos a un carrito de compras y completar una compra simulada. Está pensado como una solución sencilla pero funcional para aprender y aplicar conceptos básicos de desarrollo web, incluyendo diseño responsivo, manejo de datos dinámicos y almacenamiento local. La aplicación está construida principalmente con HTML, CSS y JavaScript, aprovechando librerías y herramientas modernas como Bootstrap para el diseño visual y SweetAlert2 para mostrar mensajes interactivos al usuario. Además, utiliza una API externa para cargar los productos de manera dinámica, lo que permite simular un entorno real de comercio electrónico sin necesidad de crear una base de datos personalizada.
